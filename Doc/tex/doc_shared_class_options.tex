The following options (and their values) will be forwarded to \pkg{MRTfonts}
(see \autoref{sec:fonts:options} for their description):
\begin{itemize}
  \item \opt{sfacc}
  \item \opt{font}
  \item \opt{serif font}
  \item \opt{mono font}
  \item \opt{new maths}
  \item \opt{scale maths}
\end{itemize}
