\chapter{The \pkg{MRTlmscale} package}\label{sec:lmscale}
This package provides the option to scale the maths font of \pkg{lmodern} in a
similar way \pkg{helvet} is scalable. It is only meant to be used with the
\hologo{pdfTeX} engine.

If you load it it applies the default scale of \num{1.17647}. This scale leads
to a matching height of the Greek maths font compared to \pkg{helvet} when used
in combination with \pkg{mathastext}. You can provide any other scale as a
package option. The package only takes this one option and it is checked whether
this option is a valid float with \cs{MRTifFloatTF}.
